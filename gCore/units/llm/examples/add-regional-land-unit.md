# Add A Regional Land Unit

Use a later-loaded project module to add a Unit with the AREA vector `{ LENGTH: 2 }`, then add one or more exact conversions scoped by country, subdivision, or locality. Add tests proving the intended geography selects exactly one rule and adjacent geographies do not.

Do not define one universal factor for a traditional name whose legal size changes by location. Do not create a second land conversion service in a real-estate or agriculture module.
