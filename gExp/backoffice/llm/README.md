# BackOffice LLM Context

Read the root and `gExp` agent contracts before changing BackOffice. Treat this
module as a backend registry/discovery capability for the separate Nodics Axis
frontend, not as a frontend or universal proxy.

Use source schemas, routers, services, facades, controllers, pipelines,
interceptors, layered properties, and existing runtime governance. Generated
context belongs under `generated/` and must not be edited manually.
