# Application Template

This directory is the scaffold template for a Nodics application layer. It provides the baseline file layout that project/application modules can copy before adding capability-specific configuration and code.

Keep this template generic. It should demonstrate the expected structure for `package.json`, `nodics.js`, and configuration files without embedding customer, tenant, or environment-specific behavior.

When framework module contracts change, update this template so newly generated applications start with the current extension model and documentation expectations.
