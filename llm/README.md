# Nodics Root AI Context

This folder contains repository-root AI context for the Nodics package boundary.

Root AI behavior is governed by `AGENTS.md`. Detailed reusable AI governance,
contracts, standards, prompts, and generated module context live under
`gSetup/llm`.

Use this folder only for root package context that does not belong to a specific
module. Prefer module-local `llm/` folders for module-specific guidance.
