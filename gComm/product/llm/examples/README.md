# Product Extension Examples

- [Extend a Product Item type](extend-product-item-type.md): add a
  project-owned type through layered configuration without duplicating Product
  or adjacent authorities.
- [Extend Product classification](extend-product-classification.md): add
  declarative policy and schema metadata while preserving typed-value and Units
  contracts.
- [Extend Product variant policy](extend-product-variant-policy.md): permit a
  project Item type or adjust bounded limits without creating another variant
  or Attribute authority.
- [Extend Product composition and publication](extend-product-composition-and-publication.md):
  customize relation/media policy or Online projection hooks while preserving
  Units, Workflow, `nPublish`, and transport authorities.
