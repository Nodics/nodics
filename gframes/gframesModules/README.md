# gframesModules

`gframesModules` groups the custom capability modules for the `gframes` application.

Use this group for project module composition and shared project-module configuration. Concrete behavior belongs in child modules such as `gfcore`, `gfprofile`, `gfcms`, and `gfapi`.

Project modules should extend framework capabilities through schemas, routers, services, pipelines, data, tests, and layered configuration.
