# gfprofile Module

`gfprofile` owns profile-related behavior for the `gframes` application. It provides project-specific profile data, schemas, routes, pipelines, interceptors, utilities, configuration, and tests.

Use this module for profile extensions that are specific to `gframes`. Shared profile capability behavior belongs in the reusable profile modules.

Profile changes must preserve privacy, validation, tenant isolation, access control, diagnostics, and generated contract alignment.
